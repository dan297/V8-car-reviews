## Installation

- install the dependencies: \
  `npm i`

- create the database with root access: \
  `mysql -u root -p` \
  `<enter password>`

- in MySQL: \
  `source db/schema.sql` \
  `quit`

- seed database: \
  `npm run seed`

## Usage

`npm start`
